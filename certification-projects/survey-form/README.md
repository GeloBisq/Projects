# Survey Form

## Overview
Survey Form is a responsive web form built as part of the freeCodeCamp Responsive Web Design certification. The project follows a set of user stories and tests to ensure proper structure, accessibility, and functionality.

The form was designed with a personalized layout and styling while maintaining compliance with the required specifications.

## Features
- Page title and description
- Name, email, and number input fields
- HTML5 input validation
- Dropdown menu
- Radio button group
- Checkbox selections
- Textarea for additional comments
- Submit button

## Technologies Used
- HTML

## User Stories Implemented
- h1 element with id="title"
- p element with id="description"
- form with id="survey-form"
- Required name and email fields
- Valid email format enforcement
- Number input with min and max validation
- Associated labels with correct ids
- Placeholder text for inputs
- Dropdown select menu
- Radio button group
- Multiple checkboxes with value attributes
- Textarea for comments
- Submit button with id="submit"

## What I Learned
- Structuring accessible HTML forms
- Using built-in HTML5 validation
- Applying CSS for layout and styling
- Translating requirements into functional UI
