# TigerTracks

## Overview
TigerTracks is an AI-powered application designed to help Riverside City College (RCC) students plan class schedules that align with their transfer goals. The system assists students in identifying courses that satisfy major requirements while fitting within their academic timelines.

This project focuses on applying AI to improve academic planning and student success.

## Features
- Collects student transfer goals
- Suggests relevant courses
- Helps build semester-by-semester plans
- AI-assisted recommendations
- User-friendly interface

## Technologies Used
- Google AI Studio

## Concepts Applied
- Prompt engineering
- AI-assisted decision support
- Data-driven planning
- User-centered design
