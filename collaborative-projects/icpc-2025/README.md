# ICPC North America Southern California Regionals 2025

## Overview
This project contains collaborative solutions developed during the ICPC North America Southern California Regional Contest (2025). It reflects real-time competitive programming under time constraints and team-based strategy.

Completed solutions are provided for Problems 1, 7, and 9, along with attempted solutions for Problems 11 and 12.
